Semantic Web Reasoning With N3

Welcome

Welcome to the online course Semantic Web Reasoning With N3.

What you will read about

  • The Semantic Web helps machines understand information on the Web.
  • RDF is the Semantic Web language.
  • Notation3 or N3 is a simple, more expressive RDF superset.

What you will learn

  • Writing RDF triples and rules in N3.
  • Using a reasoner to deduce facts.

What you need to have

  • A Web browser to read the tutorial and play with the examples online. (you have this)
  • The EYE software if you want to play with the examples offline. (not required)
